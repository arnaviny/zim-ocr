Diagram Explanation

This diagram illustrates the deployment architecture for the ZIM OCR system:

  • Frontend Container: Contains the React application served by Nginx, packaged in a Docker container.
  • Backend Container: Hosts the API and business logic (implemented using Spring Boot or Node.js) in a separate Docker container.
  • Database Container: Contains the PostgreSQL database which stores the Document data, running in its own Docker container.
  • Communication: Users access the application via a web browser; the frontend sends REST API requests (secured by JWT) to the backend, which interacts with the database.
